# Wallet Graph

wallet-graph is a prototype for analyzing and visualizing relationships between cryptocurrency wallets. It uses a graph database to store wallet connections and a web interface to explore the graph.

!!! VERY ALPHA SOFTWARE

## API

Run the API with:

```bash
cd api
go run .
```

## Web App

Web app is a Vite + Ract app:

```bash
cd web
npm run dev

# or build with
npm run build
```
